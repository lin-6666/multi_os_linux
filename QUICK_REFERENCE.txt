🚀 快速开始
═══════════════════════════════════════════════════════════

方法 1 - 完整构建 ISO:
  $ ./build-full-system.sh all
  (需要安装: xorriso genisoimage syslinux cpio mksquashfs)

方法 2 - 分步构建:
  $ ./build-full-system.sh prepare   # 准备根文件系统
  $ ./build-full-system.sh kernel     # 下载内核
  $ ./build-full-system.sh config     # 创建配置
  $ ./generate-iso.sh                 # 生成 ISO

方法 3 - Docker 构建:
  $ docker run -it debian:stable bash
  $ apt-get install build-essential xorriso
  $ ./build-full-system.sh all

💾 安装到电脑
═══════════════════════════════════════════════════════════

1. 生成完整 ISO (见上方)
2. 刻录到 USB:
   $ sudo dd if=multi-os-linux.iso of=/dev/sdX bs=4M
3. 从 USB 启动
4. 运行安装程序:
   $ mos-install
5. 重启并享受！
